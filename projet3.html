Création d'un jeu en tour par tour en java

Objectifs

Le but général de ce projet était de créer un jeu de plateau dans lequel deux châteaux (les bleus et les rouges) entraînent chacun une équipe de guerriers de différents types (chef elfe, elfe, chef nain, nain) et les envoient au combat sur un plateau de jeu constitué d’une bande de carreaux. Ce projet personnel a été développé en Java pendant une dizaine d'heures.

Description/Organisation

La première étape a consisté à créer une classe abstraite Guerrier permettant de définir les classes des différents personnages du jeu : Nain, Elfe, Chef-Nain et Chef-Elfe.

La deuxième étape a permis de concevoir les châteaux et le plateau pour y intégrer les fonctions de formations de guerriers, des combats, du moteur de jeu.

Chaque fonction développée a été testée de manière unitaire pour garantir le fonctionnement du jeu.

Techniques et savoir faire acquis

  • Bases en Java
  • Utiliser des patrons de conception pour le développement d'applications cohérentes
  • Elaborer des conception simples
  • Vérifier et valider la qualité de l'application par les tests
